STEP 1 - Set-up Project
24-48 Hrs

Create Project

Request Additional Quota

40 Min
STEP 2 - Set-up Infrastructure

Clone SNEL repository

| git clone https://github.com/snel-repo/autoLFADS-beta.git

Create server machine

| sh server_setup.sh tutserver us-central1-c

Create client machines

| sh machine_setup.sh tutcliente 4 us-central1-c
| sh machine_setup.sh tutclientc 4 us-central1-c

Check Docker finished installing

| sh check.sh pbtclient

Set-up storage infrastructure

5 Min
STEP 3 - Add User

Add user to Docker group

| sh add_docker_user.sh pbtclient

Clone SNEL repository on server

| git clone https://github.com/snel-repo/autoLFADS-beta.git
5 Min
STEP 4 - Upload Data

Upload data with prefix lfads

2 Hrs
STEP 5- Set-up Run Parameters, Start Run

Link pbt_script_multiVM.py to your data and
edit parameters

Begin autoLFADS run in tmux

| tmux
| python2 pbt_script_multiVM.py
